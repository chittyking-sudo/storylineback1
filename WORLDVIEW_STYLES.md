# 世界观风格指南

## 📚 世界观类型说明

本系统支持多种预设的世界观风格，每种风格都有专门优化的生成策略，帮助你创建更加专业和契合的内容。

---

## 🎬 电影剧本类

### 1. 喜剧 (Comedy)

**适用场景**: 轻松娱乐、搞笑桥段、情景喜剧风格

**细分类型**:
- **浪漫喜剧**: 爱情与幽默结合，温馨浪漫
- **荒诞喜剧**: 夸张的情节，超现实的笑料
- **黑色幽默**: 讽刺社会，以幽默包装严肃话题
- **情景喜剧**: 日常生活场景，角色互动为主
- **闹剧**: 夸张表演，滑稽动作，快节奏

**世界观特点**:
- ✅ 轻松愉快的基调
- ✅ 富有戏剧性的场景设定
- ✅ 角色性格鲜明，易产生笑点
- ✅ 适合日常或稍显夸张的环境

**生成优化**:
- 注重角色互动和对话幽默感
- 场景设定便于制造喜剧效果
- 背景轻松不压抑

**使用示例**:
```
项目名称: 时光旅行的爱情喜剧
世界观风格: 电影剧本-喜剧 > 浪漫喜剧
主题: 一个不擅长表达的程序员意外获得时间倒流能力，试图改变自己在女神面前的尴尬瞬间
```

---

### 2. 解谜 (Mystery/Puzzle)

**适用场景**: 推理剧情、谜题设计、真相探寻

**细分类型**:
- **推理解谜**: 逻辑推理，线索收集
- **密室逃脱**: 封闭空间，机关谜题
- **宝藏寻找**: 冒险元素，历史秘密
- **真相调查**: 案件调查，证据分析
- **谜题冒险**: 探索与解谜结合

**世界观特点**:
- ✅ 充满线索和隐藏信息
- ✅ 复杂的背景故事
- ✅ 多层次的真相揭示
- ✅ 神秘氛围营造

**生成优化**:
- 构建有逻辑漏洞待填补的背景
- 设置神秘事件和未解之谜
- 角色各有秘密和动机

**使用示例**:
```
项目名称: 古堡的秘密
世界观风格: 电影剧本-解谜 > 密室逃脱
主题: 五位陌生人被邀请到偏僻的古堡，发现自己被困，必须解开房间中的谜题才能逃生
```

---

### 3. 悬疑 (Suspense/Thriller)

**适用场景**: 紧张刺激、心理博弈、惊悚氛围

**细分类型**:
- **犯罪悬疑**: 犯罪案件，追凶过程
- **心理悬疑**: 心理层面的恐惧和怀疑
- **惊悚悬疑**: 恐怖元素，悬念迭起
- **侦探悬疑**: 侦探破案，推理为主
- **超自然悬疑**: 灵异元素，未知恐惧

**世界观特点**:
- ✅ 阴暗压抑的氛围
- ✅ 充满不确定性和危险
- ✅ 复杂的人物关系
- ✅ 隐藏的真相和动机

**生成优化**:
- 营造紧张悬疑的环境
- 角色背景复杂，动机不明
- 事件层层递进，扣人心弦

**使用示例**:
```
项目名称: 消失的记忆
世界观风格: 电影剧本-悬疑 > 心理悬疑
主题: 主角从昏迷中醒来，发现自己失去了最近三个月的记忆，而周围的人都说他做了可怕的事情
```

---

### 4. 言情 (Romance)

**适用场景**: 爱情故事、情感纠葛、浪漫情节

**细分类型**:
- **都市言情**: 现代都市背景，职场与爱情
- **古装言情**: 古代背景，宫廷或江湖
- **奇幻言情**: 魔法世界，跨越种族的爱情
- **虐恋情深**: 痛苦的爱情，波折重重
- **甜宠轻松**: 温馨甜蜜，轻松愉快

**世界观特点**:
- ✅ 浪漫美好的环境
- ✅ 适合情感发展的场景
- ✅ 角色情感丰富细腻
- ✅ 社交活动和邂逅机会

**生成优化**:
- 创造浪漫的相遇场景
- 设置情感发展的障碍和转折
- 角色性格互补或对立

**使用示例**:
```
项目名称: 星河之约
世界观风格: 电影剧本-言情 > 奇幻言情
主题: 在能够穿越星际的未来世界，一位地球女孩与来自遥远星球的王子相爱，但两个文明的战争即将爆发
```

## 🎮 游戏世界类

### 5. 魔幻游戏 (Fantasy Game)

**适用场景**: 奇幻冒险、魔法世界、史诗故事

**细分类型**:
- **RPG角色扮演**: 角色成长，技能养成
- **开放世界**: 自由探索，任务丰富
- **MMORPG**: 多人在线，社交系统
- **回合制策略**: 战术布局，策略对抗
- **动作冒险**: 即时战斗，快节奏

**世界观特点**:
- ✅ 完整的魔法体系
- ✅ 多种族和职业系统
- ✅ 史诗级的冒险背景
- ✅ 丰富的传说和神话

**生成优化**:
- 构建详细的魔法规则
- 设计种族特征和关系
- 创造史诗级的威胁和挑战
- 包含宝物、遗迹等冒险元素

**使用示例**:
```
项目名称: 永恒之光
世界观风格: 游戏世界-魔幻游戏 > 开放世界
主题: 在一个魔法与科技融合的世界，黑暗势力苏醒，玩家需要收集散落的神器碎片，重铸传说中的永恒之光
```

---

### 6. 乙女游戏 (Otome Game)

**适用场景**: 女性向恋爱模拟、多结局选择、角色攻略

**细分类型**:
- **现代都市**: 都市背景，职场或校园
- **古风仙侠**: 古代仙侠世界，修仙与爱情
- **校园青春**: 学校生活，青春恋爱
- **奇幻冒险**: 奇幻世界，冒险与浪漫
- **民国复古**: 民国时期，复古浪漫

**世界观特点**:
- ✅ 浪漫梦幻的氛围
- ✅ 精心设计的邂逅场景
- ✅ 多样化的男性角色
- ✅ 情感发展的多种路线

**生成优化**:
- 创造适合恋爱发展的环境
- 设计多个性格迥异的可攻略角色
- 安排情感升温的关键场景
- 构建选择和分支系统

**使用示例**:
```
项目名称: 繁星学院
世界观风格: 游戏世界-乙游 > 校园青春
主题: 作为转学生进入贵族学院，邂逅学生会长、孤傲天才、温柔学长等不同类型的男生，在校园生活中发展恋情
```

## 🎨 自定义风格

### 完全自定义

**适用场景**: 独特创意、混合类型、实验性设定

**特点**:
- ✅ 完全自由发挥
- ✅ 不受预设框架限制
- ✅ 可混合多种元素
- ✅ 适合创新性项目

**使用建议**:
在"主题描述"中详细说明你的设定，包括：
- 世界类型（科幻、奇幻、现实等）
- 时代背景（古代、现代、未来）
- 核心元素（魔法、科技、超能力等）
- 氛围基调（黑暗、明亮、中性）
- 独特设定（任何与众不同的要素）

**使用示例**:
```
项目名称: 赛博仙侠
世界观风格: 自定义
主题: 在一个高科技与古代仙侠文化融合的未来世界，修仙者使用纳米技术修炼，法宝由AI控制，玩家在虚拟与现实之间寻找永生的秘密
```

## 💡 选择建议

### 如何选择合适的世界观风格？

#### 1. 根据目标受众
- **年轻女性** → 乙女游戏、都市言情
- **推理爱好者** → 解谜、悬疑
- **游戏玩家** → 魔幻游戏
- **轻松娱乐** → 喜剧

#### 2. 根据内容性质
- **需要完整体系** → 魔幻游戏
- **注重情感** → 言情、乙游
- **注重逻辑** → 解谜、悬疑
- **轻松趣味** → 喜剧

#### 3. 根据创作经验
- **初次使用** → 建议选择预设风格
- **有明确想法** → 可选自定义
- **实验性质** → 可多模型对比

## 📝 使用技巧

### 1. 组合使用
可以在主题描述中结合多个元素：
```
世界观风格: 电影剧本-悬疑 > 犯罪悬疑
主题: 融合魔幻元素的现代犯罪调查，侦探拥有预知梦的能力
```

### 2. 细节补充
即使选择了预设风格，也可以在主题中添加具体细节：
```
世界观风格: 游戏世界-魔幻游戏 > RPG角色扮演
主题: 北欧神话背景，寒冷的冰雪世界，玩家扮演诸神黄昏后幸存的战士
```

### 3. 多模型对比
对于关键项目，建议使用多模型生成不同版本：
- **OpenAI GPT-4o-mini**: 均衡实用
- **Google Gemini**: 创意独特
- **OpenAI GPT-4o**: 质量最高
