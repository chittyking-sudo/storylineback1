### 配色方案

**主色调**:
- 🎨 底色: #f0d7b7 (暖米色) - 温暖、怀旧
- 💗 强调色: #e66285 (玫瑰粉) - 活力、突出
- 🌲 次要色: #2c9560 (深绿色) - 稳定、专业
- ⬛ 边框: #000100 (纯黑) - 清晰、复古

**应用场景**:
```css
/* 背景使用 */
background: #f0d7b7;

/* 主要按钮和标题 */
background: #e66285;
border: 3px solid #000100;
box-shadow: 4px 4px 0px #000100;

/* 次要按钮和卡片 */
background: #2c9560;
border: 2px solid #000100;
box-shadow: 3px 3px 0px #000100;

/* 所有边框 */
border-color: #000100;
```

### 交互效果

#### 按钮效果
```
正常状态:
┌─────────────┐
│  按钮文字   │  ← 3px 边框
└─────────────┘
    ▓▓▓▓▓▓▓▓▓    ← 4px 阴影

┌─────────────┐
│  按钮文字   │  
└─────────────┘
  ▓▓           ← 2px 阴影（向右下移动）
```

#### 悬停效果
- 按钮：背景颜色加深
- 卡片：轻微放大（scale 1.05）
- 社交图标：360° 旋转 + 放大

#### 切换效果
- 导航切换：淡入淡出动画
- 内容加载：从下往上滑入
- 地址栏：文字淡出淡入更新

### 响应式设计

#### 桌面端（>1200px）
- 完整双栏布局
- Agent 卡片 2x2 网格
- 侧边栏固定宽度 260px

#### 平板端（768px - 1200px）
- 单栏布局，创建表单和 Agent 信息上下排列
- Agent 卡片保持 2x2 网格
- 侧边栏可折叠

#### 移动端（<768px）
- 垂直堆叠布局
- Agent 卡片变为单列
- 侧边栏变为汉堡菜单

## 对比旧版本

### v2.2 → v2.3 变化

| 特性 | v2.2 | v2.3 |
|------|------|------|
| 页面数量 | 2个独立页面 | 1个统一界面 |
| 配色 | 浅米+亮粉+亮绿 | 暖米+玫瑰粉+深绿 |
| 布局 | 传统居中 | 侧边栏+双栏 |
| 导航 | 页面跳转 | 单页切换 |
| 表单位置 | 主页居中 | 右侧固定 |
| 项目列表 | 独立页面 | 同页底部 |

### 改进点

1. **更好的信息架构** - 功能分区更清晰
2. **更流畅的体验** - 无需页面跳转
3. **更统一的风格** - 配色和设计语言一致
4. **更高的效率** - 创建和查看在一个页面完成

## 用户流程示意

```
用户进入 → 浏览 Hero → 左侧选择"创建项目"
                ↓
        填写表单（右侧）
                ↓
        选择模型（复选框）
                ↓
        点击"开始生成"
                ↓
        Loading 动画（覆盖层）
                ↓
        生成完成提示
                ↓
        左侧选择"项目列表"
                ↓
        查看新创建的项目（下方）
                ↓
        点击"查看"按钮
                ↓
        新标签页打开详情页（复古风格）
```
